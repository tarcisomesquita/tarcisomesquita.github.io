Error Codes returned by USB Core Functions

  • usb_register():

    -EINVAL Error during registering new driver.

  • usb_terminate_bulk():

    -ENODEV URB already removed.

  • usb_get_*/usb_set_*():

    All USB errors (submit/status) can occur.
