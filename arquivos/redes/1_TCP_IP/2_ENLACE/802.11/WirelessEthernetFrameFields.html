802.11 Wireless Ethernet Frame Fields



Purpose of each MAC Address Field
MAC Address 1 Transmitter MAC Address
MAC Address 2 Receiver MAC Address
MAC Address 3 Wired/Infrastructure Host or Router MAC Address.
This is either the final destination or the original source of a wireless packet.
MAC Address 4 Not used in Infrastructure Wireless LANs (only used in ad hoc wireless LANs)


What is in each MAC Address Field Depends on the Direction of Travel
From wireless host to AP
Address 1 transmitter wireless host SRC
Address 2 receiver wireless AP BSSID
Address 3 infrastructure node wired host/router DST
From AP to wireless host
Address 1 transmitter wireless AP BSSID
Address 2 receiver wireless host DST
Address 3 infrastructure node wired host/router SRC


Frame Control Field subfields
Protocol Version Not used, always 0.
Type 00 Management
01 Control
10 Data
11 Future/Not Used
Subtype Management Subtypes
  • Association Request
  • Association Response
  • Reassociation Request
  • Reassociation Response
  • Probe Request
  • Probe Response
  • Beacon
  • Disassociation
  • Authentication
  • Deauthentication
Control Subtypes
  • RTS
  • CTS

Data Subtype
  • Data (duh!)
To AP & From AP Only used in ad hoc wireless,
