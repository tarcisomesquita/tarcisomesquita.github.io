Vetores podem ser usados tanto para tipos de dados primitivos quanto para classes.

Arrays são criados utilizando-se a palavra reservada new. É possível criar arrays de arrays.

Exemplos:
// declarando
int[] Vet;
Vet = new int[12]; ou int [] Vet = new int[12]; ou int Vet[] = new int[12];

// declarando e inicializando
int [ ] Vet2 = { 0, 6, 12, 18, 24, 30, 36, 42, 48, 54, 60, 66};


// declarando
int[ ][ ] Mat;
Mat = new int [5] [2]; ou int Mat[ ][ ] = new int [5] [2];
