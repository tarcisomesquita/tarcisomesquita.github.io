
Control operators:
& && ( ) ; ;; | || <newline>


Redirection operators:
< > >| << >> <& >& <<- <>

read TEXTO << DELIMITADOR
meu texto
DELIMITADOR
equivale a
read TEXTO <<< "meu texto"
<< é pra multilinha
<<< é para uma única linha

&& equivale a AND

|| equivale a OR

$ ping -c1 www.linux.org && echo 'rede OK' || echo 'rede SUJOU!'

Só executa o terceiro comando se os dois primeiros não apresentarem erro.
