Note: There are two options for changing the browser Geolocation.

a. Fill out the below table with desired values for latitude, longitude, etc.

b. Use the interactive map at: https://webbrowsertools.com/geolocation/?zoom=3

To use the map, please find a desired location and then press - Send to your Geolocation Extension - button.
Geolocation Metrics (default location is Greenwich, UK)
1. Coordinate altitude (geographical position altitude in meters)
2. Coordinate heading (direction in which the device is traveling)
3. Coordinate speed (device velocity in meters per second)
4. Coordinate latitude (geographical position latitude in decimal degrees)
5. Coordinate accuracy (the accuracy, with a 95% confidence level in meters)
6. Coordinate longitude (geographical position longitude in decimal degrees)
7. Coordinate altitudeAccuracy (the accuracy, with a 95% confidence level in meters)
8. Timestamp (the time at which the location was retrieved in milliseconds)
