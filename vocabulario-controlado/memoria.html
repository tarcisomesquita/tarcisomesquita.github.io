Memória

Descrição

Uma memória é um objeto composto de partes diferentes.

Exemplo

  • Uma fita de papel. Cada pedaço é uma parte. Alguns pedaços furados, outros não.
  • Uma fita de material magnético. Cada pedaço é uma parte. Alguns pedaços magnetizados, outros não.
  • Uma onda. Cada pulso é uma parte. Alguns pulsos com uma amplitude, outros com outra.
  • Uma onda segmentada em trechos com frequências diferentes.
  • A luz emitida por LED piscando. Neste caso, uma variação temporal do LED gera uma variação espacial (onda de luz). Obs.: uma tela pode ser formada por um conjunto de LEDs.
